## 4. ตรวจสอบระบบกรองเสียง / Noise Filtering / VAD + ความแม่นยำของ Speech-to-Text

Domain: `internal/audio`, `internal/vad`

- [ ] ทดสอบ streaming RMS-VAD (Stage 08) ด้วยไฟล์เสียงจริงที่มี noise
      (เช่น เสียงพัดลม, เสียงพิมพ์คีย์บอร์ด) — ตรวจว่าตัด silence/noise ออก
      ได้ถูกต้อง โดยไม่ตัดเสียงพูดทิ้ง
- [ ] ตรวจ threshold ของ `IsLikelyHallucination` (Stage 09) ว่ายังเหมาะสมกับ
      เสียงที่มี noise หรือไม่ — ปรับค่าถ้าจำเป็น พร้อมบันทึกเหตุผลใน
      DECISIONS.md
- [ ] ทดสอบ end-to-end ด้วยเสียงสภาพแวดล้อมจริง (มีเสียงรบกวนรอบข้าง)
      เทียบ transcript ที่ได้กับคำพูดจริง — วัด accuracy
- [ ] ทดสอบ edge case: input เงียบสนิท (ไม่มีเสียงพูดเลย) → ต้องไม่ panic,
      ไม่ส่ง chunk ว่างไป transcribe API (ตาม Stage 03/08 acceptance criteria)
